U Unearth Design System: Icons
UI Icons
Application UI icons can be rendered with any of the following options to optimize client efficiency.
Unearth Icons 3.0 (OPv2 latest)
  • font files
  • SVG
  • PNG
Unearth Icons 2.4 (OPv1 latest)
  • font files
  • SVG
  • PNG
Asset Icons
In order to support user-uploaded visuals in the future, assetType and featureType icons are implemented as PNGs stored in S3. Full path: https://media.unearthlabs.com/{mediaID}
assetType:
  • used for tools, cards, asset forms, point menus, filters, layers menu
  • @2x display size for higher density displays
featureType:
  • used on the map
  • geometryType = point with fixed pixel size marker (i.e. does not scale with map): @2x display size
  • geometryType = geo-sized point (i.e. scales with the map): width and/or height = 700px
  • geometryType = polygon: does not apply
  • geometryType = polyline: does not apply
  • assetType = Text, Plan, Geo Data, or UAV Survey: does not apply
