U Unearth Design System: Forms
Forms
Container:
  • Background color should contrast with form components, e.g. $slate5
  • Max-width may vary depending on context, but in general min-width should not be less than 280px
  • Minimum padding of 24px within container helps to minimize cramping
  • If height of form exceeds viewport height, submit button should pinned to bottom and form content scrolls behind
Inputs:
  • Label details:
    • 13px/1/500 Museo Sans Rounded $slate2
    • letter-spacing: 0.25px
    • margin-bottom: 4px
    • short label can be inset to conserve vertical space
  • Input details:
    • 16px/1/500 Museo Sans Rounded $slate2
    • placeholder text: $slate4
    • optional tip text: 13/1/300, margin-top: 4px
    • if input fails validation onBlur, display error state (label and border color: $red1) and message (replaces tip text if any)
Selects:
  • Label details:
    • 13px/1/500 Museo Sans Rounded $slate2
    • letter-spacing: 0.25px
    • margin-bottom: 4px
  • Select details:
    • 16px/1/500 Museo Sans Rounded $slate2
    • placeholder text: $slate4
    • use chevron-down-circle icon as select control affordance
    • custom select options can include icons
Segmented Controls:
  • Control Details:
    • Inset Label: 16px/1/500 Museo Sans Rounded $slate3
    • background-color: $slate, box-shadow: inset 0 0 12px rgba(0,0,0,0.075), inset 0 0 1px 1px rgba(0,0,0,0.05)
    • Active State: background-color $teal1
Text Areas:
  • Label details:
    • 13px/1/500 Museo Sans Rounded $slate2
    • letter-spacing: 0.25px
    • margin-bottom: 4px
  • Text Area details:
    • 16px/1/500 Museo Sans Rounded $slate2
    • placeholder text: $slate4
Checkboxes:
  • Checkbox details:
    • width 20px; height 20px; border-radius 2px;
    • Tap Target should wrap label as well
  • States:
    • Active/On: border-color $teal1; background-color $teal2; content: icon-check; font-size 10px;
    • Inactive/Off: border-color $slate3; background-color $arctic1; box-shadow inset 0 0 6px rgba(0,0,0,0.05)
    • Indeterminate: border-color $teal1; background-color $teal2; content: -
  • Label details:
    • 14px/1/500 Museo Sans Rounded $slate2
    • margin-left: 8px;
Buttons:
  • Form buttons should never be out of the viewport, so wrap them in a form footer that is pinned to the bottom of the container
  • Form footer should have visual contrast with form container, background-color: arctic2, box-shadow serves as a scrolling affordance
  • Single Action:
    • Primary button styling
    • center align button in form footer
  • Primary/Secondary button pair:
    • Constructive action on the right: primary button styling
    • Destructive or Cancel action on the left: secondary button styling
